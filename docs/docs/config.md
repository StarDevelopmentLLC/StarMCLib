# Config class
This is a wrapper for [BoostedYaml](https://www.spigotmc.org/threads/%E2%9A%A1boostedyaml-feature-rich-library-write-once-run-everywhere-updater-comments-and-more-%E2%9A%A1.545585/) YamlDocument.  
This wraps the file-based methods and catches the exceptions in addition to adding `addDefault` methods that allow specifying default values.  
These defaults are kind of similar to spigot's default options and allow you to add comments in a single line of code and with comments.
